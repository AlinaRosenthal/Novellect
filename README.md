# 📚 Novellect: Intelligent Local Library & Agentic RAG

**Novellect** — это интеллектуальная система управления локальной библиотекой книг. В отличие от обычного поиска, Novellect использует **гибридный подход** (BM25 + нейросетевые эмбеддинги) и **локальных ИИ-агентов**, позволяя находить информацию по смыслу и получать обоснованные ответы без использования внешних API.

Проект поддерживает два режима работы: от «легкого» поиска для слабых устройств до полноценного агентского пайплайна с локальной языковой моделью (LLM).

---

## ✨ Основные возможности

*   **Гибридный поиск (Hybrid Search):** Сочетание алгоритма BM25 (поиск точных слов) и семантических векторов (поиск по смыслу) через модель `paraphrase-multilingual-MiniLM-L12-v2`.
*   **Два режима работы (Lite & Full):** 
    *   **Lite:** Быстрый поиск с ранжированием, идеально для CPU и слабых ПК.
    *   **Full:** Агентный RAG с использованием локальной LLM для планирования и синтеза ответов.
*   **Семантическое разбиение (Semantic Chunking):** Интеллектуальное деление текста на блоки с сохранением контекста повествования.
*   **Локальная LLM без API:** Полная приватность. Система использует локальные модели (например, Qwen 2.5) через `transformers`.
*   **Дообучение (Domain Adaptation):** Возможность адаптировать embedding-модель под свою библиотеку через **LoRA** или полный Fine-tuning прямо в интерфейсе.
*   **Мультиформатность:** Поддержка `.pdf`, `.fb2`, `.epub`, `.txt` и автоматическая распаковка `.zip`.
*   **Импорт из облака:** Поддержка прямых ссылок, Google Drive и Google Cloud Storage.

## 🏗 Архитектура системы

### 1. Конвертация и обработка (`converter.py`, `archive_handler.py`)
*   **Парсинг:** Извлечение текста и метаданных (названия, авторы) из различных форматов.
*   **Архивы:** Безопасная работа с ZIP-файлами без полной распаковки на диск.
*   **Кэширование:** Система сохраняет текстовые копии тяжелых документов для мгновенного повторного доступа.

### 2. Поисковой движок (`search_engine.py`)
*   **Векторное хранилище:** Локальная база на базе NumPy (`vector_db.npz`), обеспечивающая переносимость и скорость.
*   **Semantic Similarity:** Использование косинусного сходства для поиска релевантных фрагментов.
*   **Reranker:** Интегрированный Cross-Encoder для финальной перепроверки топ-результатов.

### 3. Агентный слой (`agents.py`, `llm_runtime.py`)
Система построена на взаимодействии специализированных агентов:
*   **QueryAnalyzerAgent:** Определяет намерение пользователя (поиск книги, вопрос по сюжету или рекомендация).
*   **LLMPlannerAgent (Full mode):** Строит стратегию поиска, разбивая сложный вопрос на несколько подзапросов.
*   **GroundedAnswerAgent (Full mode):** Формирует финальный ответ, опираясь **только** на найденные в библиотеке источники (с цитатами и ссылками).

## ⚙️ Режимы работы и требования

| Режим | Описание | Требования (RAM/VRAM) |
| :--- | :--- | :--- |
| **Lite** | Стабильный поиск (BM25 + Embeddings + Rerank). Без генерации текста. | 2–4 ГБ RAM, GPU не требуется. |
| **Full** | LLM-агент строит план поиска и пишет связный ответ по источникам. | 6–10 ГБ RAM, желательно 3+ ГБ VRAM (поддержан CPU fallback). |

## 🧪 Дообучение модели (LoRA)

Novellect позволяет улучшить качество поиска по вашей специфической библиотеке:
*   **Синтез пар:** Система сама создает пары "запрос-ответ" на основе ваших книг.
*   **LoRA Adaptation:** Параметро-эффективное обучение, которое не портит базовые знания модели, но адаптирует её под домен (жанры, термины).
*   **Безопасность:** Новая модель активируется только после успешной проверки, старая сохраняется в бэкап.

## 📊 Сравнение и Целевая Аудитория

**Для кого этот проект:**
*   Студенты и исследователи, работающие с локальными базами знаний.
*   Владельцы больших цифровых архивов, желающие иметь "умного" ассистента офлайн.
*   Разработчики, изучающие архитектуру RAG (Retrieval-Augmented Generation) и ИИ-агентов.

**В чем отличие от аналогов:**
*   **Автономность:** Работает без OpenAI/Anthropic API.
*   **Гибкость:** Переключение между режимами "быстро/просто" и "умно/ресурсоемко".
*   **Grounded QA:** В отличие от обычных чат-ботов, Novellect не выдумывает ответы, а цитирует ваши книги.
