# Novellect Proto1 — Интеллектуальная RAG-система для локальной библиотеки

**Novellect** — это система семантического поиска по книгам в локальной библиотеке.  
Она использует RAG (Retrieval-Augmented Generation) для поиска по смыслу, тропам, персонажам и сюжетам в файлах форматов `.pdf`, `.epub`, `.fb2`, `.txt`.

## Возможности

- 🔍 Семантический поиск по содержанию книг
- 📚 Поддержка популярных форматов электронных книг
- ⚡ Быстрая индексация и поиск по локальной библиотеке
- 🖥️ Удобный веб-интерфейс на Streamlit
- 🧠 Поиск на естественном языке (не просто по ключевым словам)
